Decision Tables / TABULATE + FREQ Package

Project description:
A Python package for creating polished, executive-ready summary tables and categorical crosstabs from business data. It would combine the spirit of SAS PROC TABULATE and PROC FREQ with modern Python workflows.

Goal:
Make it easy to produce management-quality tables with counts, percentages, totals, subtotals, multiple statistics, weighted summaries, statistical tests, effect sizes, and clean Excel/HTML output — without hand-building pandas pivot-table origami every time.

Core positioning:

Executive-ready business tables without the spreadsheet archaeology.
